# Bun

1. Install [Bun](https://bun.sh/docs/installation)

Save to `index.ts`

```ts
{{#include ./index.ts}}
```

Save to `package.json`

```json
{{#include ./../../../package.json}}
```

You should now have the following files

- current directory
  - `index.ts`
  - `package.json`

Run the following commands:

- `bun run ./index.ts`
